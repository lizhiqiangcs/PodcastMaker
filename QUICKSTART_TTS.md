# 🚀 Azure TTS Generator - 快速开始

## 一键启动（推荐）

### macOS/Linux:
```bash
./start-tts.sh
```

### 或使用npm命令:
```bash
npm run tts
```

## 打开网页界面

启动后，打开浏览器访问：
```
http://localhost:3001/tts-generator.html
```

## 使用步骤

### 1️⃣ 选择SSML文件
- 点击 `ski-3-concepts-podcast.ssml` 文件卡片
- 卡片会高亮显示（紫色背景）

### 2️⃣ 调整语音参数（可选）
- **John声音**：默认 Guy (专业男声)
- **Alice声音**：默认 Jenny (活泼女声)
- **语速**：默认 1.0x（正常）
- **音频质量**：默认 高质量（推荐）

### 3️⃣ 试听片段
- 点击 **"🎧 试听片段 (30秒)"** 按钮
- 等待3-5秒生成
- 播放试听，验证声音效果
- 不满意可调整参数后重新试听

### 4️⃣ 生成完整音频
- 满意后，点击 **"🎵 生成完整音频"** 按钮
- 等待40-60秒（根据内容长度）
- 生成完成后显示保存路径
- 音频自动保存到 `audio-output/` 目录

### 5️⃣ 播放/下载
- 网页中可直接播放生成的音频
- 或在 `audio-output/` 目录找到MP3文件

## 💰 预估成本

- **ski-3-concepts-podcast.ssml** (~15,000字符)
- 试听生成（2000字符）：~$0.03
- 完整生成（15,000字符）：~$0.24
- **总成本不到0.3美元**

## 🎨 推荐配置

**日常播客**：
```
- John: Guy Neural (专业男声)
- Alice: Jenny Neural (活泼女声)
- 语速: 1.0x
- 质量: 高质量 (24kHz, 160kbps)
```

**快节奏对话**：
```
- 语速: 1.05x - 1.1x
```

**教学讲解**：
```
- 语速: 0.95x - 1.0x（稍慢）
```

## 📁 输出文件

生成的文件会保存在：
```
audio-output/ski-3-concepts-podcast_2026-01-21T12-00-00.mp3
```

文件命名格式：
```
<原文件名>_<时间戳>.mp3
```

## ⚠️ 注意事项

1. **首次生成可能较慢**：Azure服务初始化需要时间
2. **网络要求**：需要稳定的网络连接到Azure服务
3. **成本控制**：使用试听功能可避免不必要的完整生成
4. **并发限制**：避免同时生成多个音频

## 🔧 故障排除

### 服务器启动失败
```bash
# 检查端口是否被占用
lsof -i :3001

# 如果被占用，杀掉进程或换端口
```

### Azure认证失败
检查 `config-template.js` 中的：
- `subscriptionKey` 是否正确
- `region` 是否为 `eastus2`

### 生成失败
1. 检查网络连接
2. 查看控制台错误信息
3. 确认Azure账户有余额
4. 尝试重新启动服务器
